# 文件夹展示首页

一个自动获取并展示当前目录下所有文件夹的Web应用。

## 功能特点

- 🚀 自动扫描当前目录下的所有文件夹
- 📊 显示文件夹的详细信息（大小、创建时间、修改时间）
- 🎨 现代化的卡片式界面设计
- 📱 响应式布局，支持移动端
- 🔄 自动刷新（每30秒）

## 安装和使用

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 访问页面

打开浏览器访问：http://localhost:3000

## 项目结构

```
.
├── server.js          # 后端服务器（Express）
├── package.json       # 项目配置和依赖
├── public/           # 前端静态文件
│   ├── index.html    # 主页面
│   ├── style.css     # 样式文件
│   └── script.js     # 前端逻辑
└── README.md         # 说明文档
```

## API接口

### GET /api/folders

获取当前目录下的所有文件夹信息。

**响应示例：**
```json
{
  "success": true,
  "folders": [
    {
      "name": "文件夹名称",
      "path": "/完整路径",
      "created": "2024-01-01T00:00:00.000Z",
      "modified": "2024-01-01T00:00:00.000Z",
      "size": 1024
    }
  ]
}
```

## 注意事项

- 默认端口为 3000，如需修改请编辑 `server.js`
- 会自动排除 `node_modules` 和隐藏文件夹（以 `.` 开头）
- 文件夹大小计算是递归的，大文件夹可能需要一些时间
